# Interactive-website--for-resume
using HTML-CSS-JAVASCRIPT

Project Overview
This project is an interactive, responsive personal portfolio website for Vani Dasari, showcasing his skills, projects, education, work experience, certifications, and interests. The site is built with semantic HTML, modular CSS, and vanilla JavaScript to create a professional and accessible web presence.

Features
Responsive design suitable for desktops, tablets, and mobile devices

Sticky navigation menu with active section highlighting and smooth scrolling

Dynamic greeting message in the About Me section

Animated reveal for work experience entries with tooltips for responsibilities

Sortable education table by GPA/percentage

Skills section with toggleable beginner, intermediate, and advanced skill lists

Projects displayed as flip cards featuring relevant images and colored backgrounds; filterable by technology

Interests section with accessible list and modal popups for additional details

Certifications section highlighting key achievements with animated badges

Light and dark theme switcher with color preferences saved in localStorage

Back-to-top button appearing on scroll for easy navigation

Technologies Used
HTML5, CSS3

Vanilla JavaScript (ES6+)

How to Use
Open the index.html file in any modern web browser.

Navigate through sections using the sticky navbar.

Click or use keyboard navigation to explore details in projects and interests.

Use the theme switcher button to toggle between light and dark modes.

Sort the education table by GPA/percentage by clicking the sort button.

Filter projects by technology using the filter buttons.

Scroll down to reveal animations and the back-to-top button.

Structure
All code is contained in a single HTML file with embedded CSS and JS.

Modular CSS classes and semantic tags structure content cleanly.

JavaScript handles interactivity without external dependencies.

Customization
Replace project images (URLs) and text in the Projects section to reflect your work.

Edit skills lists to match current expertise.

Update contact information and social links as needed.
